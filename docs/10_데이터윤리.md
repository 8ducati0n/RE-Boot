**작성일**: 2026-05-14
**기준**: 개인정보 보호법(PIPA), 정보윤리 일반 원칙, IRB 일반 가이드라인
**관련 anchor**: `05_제출초록.md` (학습자 데이터 수집·분석을 전제로 함)

> 본 시스템은 학습자의 자기평가·체크리스트·CAM 대화·퀴즈 응답·이수율 등 **학습 행동 데이터**를 수집·분석한다. 본 문서는 그 수집·저장·이용·삭제 절차에 대한 윤리 정책을 1쪽 분량으로 정의한다.

## 1. 데이터 분류

| 분류 | 예시 | 식별성 | 보관 기간 (초안) |
|---|---|---|---|
| **A. 학습 행동 데이터** | 자기평가 응답, 체크리스트 증거 입력, 망각곡선 퀴즈 응답, CAM 대화 로그 | 가명·식별 가능 | 부트캠프 회기 종료 후 1년 |
| **B. 메타인지 분석 산출물** | premature_confidence, system1_demand 등 신호, Risk Score | 학습자 ID에 묶임 | 회기 종료 후 6개월 |
| **C. 강사 작성 메시지·평가** | Composition Assistant 사용 이력, 강사가 작성·발송한 메시지 | 학습자 식별 | 회기 종료 후 1년 |
| **D. 운영 메타데이터** | 로그인 시각, IP, 디바이스 | 학습자 식별 | 90일 후 익명화 |

## 2. 수집 동의 (Informed Consent)

학습자는 부트캠프 입과 시 다음 사항을 명시적으로 안내받고 동의한다.

- [ ] 수집되는 데이터 종류와 사용 목적 (분류 A–D 별 항목 안내)
- [ ] AI가 분석한 산출물이 강사에게 전달된다는 사실
- [ ] 강사가 본 데이터를 기반으로 개입(메시지·면담·자료 추천)할 수 있다는 사실
- [ ] 본인의 데이터를 **열람·정정·삭제 요청** 권리
- [ ] 동의 거부 시에도 기본 학습 콘텐츠는 동일하게 제공되며, **AI 분석·강사 개입 기능만 비활성화**된다는 사실
- [ ] 데이터 보관 기간 종료 후 자동 익명화 또는 삭제

**원칙**: 동의는 한 번에 한 묶음으로 받지 않고, 분류 A–D별 **세분화 동의(granular consent)**를 받는다.

---

## 3. AI 산출물 처리 원칙

### 3-1. 학습자 노출 금지 영역
- AI가 추출한 **메타인지 신호 (분류 B)**는 학습자에게 직접 노출하지 않는다.
  - 예: "premature_confidence" "system1_demand" 같은 라벨은 학습자 화면에 절대 등장하지 않음
- AI가 산출한 **Risk Score 등급**은 강사 대시보드에만 노출. 학습자는 본인의 점수·등급을 보지 못한다.
- 하위 25% 학습자에게는 사회비교 정보 노출이 약화된다 (`06_포스터_기획설계.md` B-2-보강 참조).

### 3-2. 강사 영역의 책임
- AI 산출물의 **해석·결정**은 강사 책임이다.
- 강사가 학습자에게 보내는 메시지는 강사가 **직접 작성**한다. AI는 사실·옵션·표현 재료만 제공.
- 모든 학습자 발송 행위는 강사 명의로 로그된다.

### 3-3. AI 모델 학습 (재사용) 금지
- 수집된 학습자 데이터는 본 시스템 내 분석·개입에만 사용된다.
- **외부 LLM(Anthropic, OpenAI 등)의 모델 학습용으로 송신·제공하지 않는다.**
- API 호출 시에도 학습자 개인 식별 정보는 추상화·가명화된 형태로만 송신한다.

---

## 4. 학습자 권리 (Data Subject Rights)

| 권리 | 행사 방식 | 응답 기한 |
|---|---|---|
| **열람권** | 마이페이지에서 본인 데이터 다운로드 (JSON·CSV) | 즉시 |
| **정정권** | 자기평가·체크리스트 본인 응답 수정 (수정 이력 보관) | 즉시 |
| **삭제권** | 마이페이지에서 "데이터 전체 삭제" 요청 | 30일 이내 처리 |
| **이의 제기권** | AI 산출물(신호·점수) 이의 제기 → 강사·운영자 검토 | 14일 이내 응답 |
| **이용 정지권** | AI 분석·강사 개입 기능 일시 정지 | 즉시 |

---

## 5. 보관·삭제·익명화

- **암호화**: 분류 A–C는 저장 시 AES-256 암호화. 분류 D는 IP 마스킹 후 저장.
- **보관 기간 만료 시**: 보관 기간이 만료된 데이터는 자동 익명화 (학습자 ID 제거) 또는 영구 삭제.
- **학습자 탈퇴 시**: 즉시 삭제. 단, 운영상 법적 보관 의무가 있는 항목(회계·세무 등)은 별도 명시 후 보관.
- **백업 정책**: 백업본도 동일한 보관 기간·암호화 정책을 따른다.

---

## 6. 미성년자 보호

- 본 시스템은 만 14세 이상을 기본 대상으로 한다.
- 미성년자 학습자는 **법정대리인 동의**를 추가로 받는다.
- 미성년자의 메타인지 신호·Risk Score는 강사에게도 **추상화된 형태(낮음/중간/높음 3단계)로만** 제공한다.
